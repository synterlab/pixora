# PIXORA

**Adventures in the Age of AI**

A premium mobile-first educational adventure game for children aged 8–13.

## Deploy

Built with React + Vite + Framer Motion. Deployed via Vercel.



## Worlds
- 🌲 Crystal Forest — Spot AI-generated images
- 📚 Echo Library — Detect AI-generated text
- 🔭 Truth Observatory — Facts vs hallucinations
- 🏙️ Future City — Responsible AI use
